Slideshow Settings

Open file /assets/js/template.js You'll find the class for every element that contain slideshow.
  1. Testimonials Slide find variable var tslide
  2. Related Articles find variable var related_articles
  3. Portfolio Detail Slide find variable var portfolio_slide
  4. Milestones Slideshow (Work Experience) find variable var milestones_slideshow
